Страница программы "Архив документов"

Конфигурация "Юридическое дело" - это CRM-система для юристов, предназначенная для учета различных дел, судебного делопроизводства юридического отдела, нотариального документооборота.

Программа содержит справочник судов. При её помощи можно регистрировать входящие и исходящие документы; формировать договоры.

Она позволяет вести учёт клиентов; участников судебного процесса; учёт исковых заявлений, претензий, жалоб; учёт доверенностей; вести календарь судебных заседаний.

Программа хранит информацию о сотрудниках, даёт возможность формировать ежемесячную зарплату для них.

Гибкая структура базы данных позволяет настраивать программу под конкретные требования заказчика



Основные функции:

  • Регистрация входящих, исходящих документов;
  • Формирование договоров;
  • Календарь судебных слушаний;
  • Справочник судов;
  • Учет клиентов (юридические и физические лица);
  • Учет участников судебного процесса;
  • Учет исковых заявлений;
  • Учет претензий и жалоб;
  • Учет доверенностей;
  • Хранение информации о сотрудниках, настройка персональных прав доступа к системе для сотрудников;
  • Возможность формирования ежемесячной заработной платы для сотрудников;
  • Гибкая структура БД с настройкой под любые задачи;
  • Возможность импорта и экспорта данных.

Установка конфигурации

Для установки и запуска необходимо:

  1. Скачать и установить программу "Архив документов" (если еще не установлена)
  2. Откопировать файл конфигурации в папку с базами данных
  3. Запустить программу "Архив документов" и открыть данную БД, выбрав в меню "Файл" -> "Открыть базу данных..."
