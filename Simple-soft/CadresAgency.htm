Страница программы "Учет клиентов"

Конфигурация "Кадровое агентство" - предназначена для автоматизации работы кадрового агентства. Программа для кадрового агентства является комплексным решением для управления бизнес-процессами, а также повышения качества и эффективности работы сотрудников кадровых агентств. Удобная система поиска кандидатов и вакансий позволит максимально сэкономить время выполнения задач. Конфигурация легко и быстро настраивается под конкретные требования заказчика.

Гибкая структура базы данных дает возможность создавать новые таблицы, отчеты, графики, добавлять поля, задавать списки и многое другое. Система учета для кадровых агентств интуитивно проста для пользователей и не требует квалифицируемых ИТ-ресурсов. Вы сможете настроить программу и на любую другую предметную область без специальных знаний.

Видео презентация

Основные функции:

Учет работодателей и их данных
Ведение базы данных работодателей и их контактов.
По каждому работодателю можно посмотреть данные по
услугам, оказанным ему, а также платежам.
Учет соискателей
Ведение базы данных соискателей со всей информацией,
важной для их трудоустройства. Учет резюме.
Учет задач
Регистрация и учет задач(заявок) на подбор персонала.
Учет вакансий
Ведение базы данных вакансий со всей информацией,
важной для соискателей.
Учет событий
Регистрация и учет запланированных и прошедших
собеседований, а также других событий. Учет результатов.
Настройка напоминаний.
Учет сделок
Регистрация и учет сделок и договоров на услуги по
подбору персонала.
Выставление счетов
Формирование счетов на оплату услуг по подбору
персонала для работодателей.
Учет платежей
Регистрация и контроль наличных и безналичных платежей.
Учет сотрудников
Хранение информации о сотрудниках, настройка
персональных прав доступа.
Система напоминаний
Сообщения об актуальных задачах, поставленных
пользователям на указанный срок.
Формирование документов по шаблонам
Набор стандартных отчетов с возможностью создания новых

Установка конфигурации

Для установки и запуска необходимо:

  1. Скачать и установить программу "Учет клиентов" (если еще не установлена)
  2. Скачать файл конфигурации, извлечь файл из архива и откопировать в папку с базами данных
  3. Запустить программу "Учет клиентов" и открыть данную БД, выбрав меню "Файл" -> "Открыть базу данных..."
