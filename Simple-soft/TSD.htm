Решение для терминала сбора данных и программы "Склад и торговля" Сделать заказ

Терминал сбора данных (ТСД) является устройством для оперативного сбора, обработки и передачи информации о товарах в базу данных. Для корректной работы устройства ТСД с номенклатурой товаров требуется 2 вида программ: операционная система самого терминала и приложение для синхронизации баз данных терминала и ПО.

Видео презентация

Для осуществления синхронизации программы Склад и торговля и терминала сбора данных предлагаем программу TsdSync.

TsdSync – программа для обмена данными между программой "Склад и торговля" и ТСД (терминалом сбора данных). TsdSync позволяет использовать ТСД при работе с программой “Склад и торговля”. Синхронизация между ТСД и базой данных программы "Склад и торговля" значительно оптимизирует работу предприятия и автоматизирует учет операций с товаром.

ТСД (Opticon Smart) + TsdSync + "Склад и торговля"

Схема работы:

  1. С помощью программы TsdSync производиться настройка соединения с ТСД;
  2. Посредством программы TsdSync в ТСД из базы данных "Склад и торговля" загружаются наименования, штрих-коды, цены и остатки товаров;
  3. Производится сканирование штрих-кодов товаров, а также различные операции над товарами при помощи ТСД;
  4. После завершения работы с ТСД при помощи программы TsdSync данные из ТСД загружаются в базу данных программы "Склад и торговля".

Основные возможности ТСД:

 Сканирование штрих-кодов, ввод новых штрих-кодов
Сканируется штрих-код товара и определяется - есть ли товар в списке на ТСД. При отсутствии товара в списке на ТСД, его можно добавить как новый.
 
 Поступление товаров
Сканируется штрих-код поступающего товара. Полученные данные отправляются в базу данных программы "Склад и торговля" и могут быть использованы для оформления приходной накладной.
 
 Продажа товаров
Продаваемый товар сканируется терминалом. Полученные данные передаются на компьютер по мере необходимости.
 
 Оформление заказов на товар
Сканируются штрих-коды образцов товара. Количество в процессе подбора заказа можно редактировать вручную. Полученные данные отправляются в базу данных программы “Склад и торговля” и могут быть использованы для оформления заказа покупателя.
 
 Списание товаров
С помощью терминала сканируются штрих-коды товаров, которые подлежат списанию.
 
 Перемещение товаров
Сканируются штрих-коды отгружаемого или перемещаемого товара. Полученные данные отправляются в базу данных программы “Склад и торговля” и могут быть использованы для оформления расходной или товаротранспортной накладных.
 
 Инвентаризация товаров
Сканируются штрих-коды товара. Количество товара в процессе проведения инвентаризации можно вводить как автоматически, так и вручную. Полученные данные отправляются в базу данных программы “Склад и торговля” и могут быть использованы для сравнения с какой-либо поставкой или состоянием склада.
 
 Проверка информации о товаре
Сканируются штрих-коды товара, находящегося в торговом зале. Информация, обозначенная на ценнике товара, сравнивается с информацией, отображаемой на дисплее ТСД. Обычно это используется для проверки актуальности цены, расположенной на ценнике товара.
 
 Удаление товара из операции
В случае занесения в операцию не того товара или неверного количества ,можно удалить его из текущей операции в указанном количестве.

Основные возможности программы TsdSync:

 Настройка соединения с ТСД
Из TsdSync можно открыть окно драйвера ТСД и осуществить настройку параметров вручную, проверку связи с ТСД, поиск оборудования или открытие мастера настроек ТСД.
 
 Загрузка из базы данных программы "Склад и торговля" в ТСД списка товаров, штрих-кодов, цен и остатков.
Программа TsdSync позволяет загрузить актуальный список товаров в ТСД с информацией о цене и количестве по каждому товару. Также можно обновить уже имеющийся список до актуального состояния.
 
 Просмотр информации, расположенной в ТСД
Имеется возможность просматривать информацию о товарах, загруженных в ТСД, информацию по всем операциям с товарами, а также по каждой операции в отдельности.
 
 Загрузка информации из ТСД в базу данных программы "Склад и торговля"
После того, как все операции с товарами завершены, и вся необходимая информация находится в ТСД, необходимо перенести ее в базу данных программы "Склад и торговля". При загрузке вся информация по операциям распределяется по соответствующим таблицам в базе данных “Склад и торговля”. В случае успешной загрузки выводиться сообщение с информацией по загруженным операция.
 
 Удаление всей информации из ТСД
В случае, если на ТСД имеется неактуальная или ненужная информация, можно удалить все данные из ТСД, после чего можно загрузить новый список товаров и продолжить работу с ТСД.
