PsSync1C Скачать

PsSync1C – это внешняя обработка 1С Предприятия для синхронизации данных между справочниками, журналами документов, документами 1С и таблицами баз данных (БД) Access/MS SQL Server.

На данный момент программа синхронизирует данные справочников, журналов документов, документов и таблиц Access/MS SQL Server, в ближайшее время список документов для синхронизации будет расширен.

ВНИМАНИЕ! Данная версия демонстрационная, в ней отсутствует функционал сохранения настроек в файл и открытие сохраненного файла настроек, а так же автоматическая синхронизация через заданный промежуток времени и имеется ограничение на синхронизируемые документы. Текущая версия находится в процессе тестирования, рекомендуется перед началом ее использования сделать резервные копии БД 1С и Access/MS SQL Server, во избежание потери данных, либо нарушения корректной работы БД.

В случае возникновения ошибок в процессе использования, просьба обращаться в "Техническую поддержку" согласно пунктам представленным на странице технической поддержки.
