Видео для скачивания

1. Запуск с ключом prolong - ролик демонстрирует запуск программы с параметром "prolong" в соответствии с типовым вопросом №1

2. Условная видимость - данный краткий видеоролик показыват условную видимость полей в действии

3. Копирование записи в другую таблицу - демонстрация переноса текущей записи главной таблицы и подчиненных записей в другую таблицу

4. Продажи в рассрочку - генерация графика платежей, которые в будущем будут погашаться клиентом. Добавление нескольких записей в подчиненную таблицу из триггера в цикле

5. Многовоалютность - демонстрация ввода суммы в разных валютах

6. Ссылка на файл-картинку - и значение по умолчанию для полей-картинок

7. Генерация расписания - конкретных занятий по датам для группы и отметка присутствующих на занятии на примере конфигурации "Обучающий центр"

8. Счетчик посещений клиентов - демонстрация работы счетчика визитов клиентов на примере конфигурации "Фитнес-клуб"

9. Добавление нового поля с формулой - инструкция по добавлению формулы в поле для его автоматического расчета. Использование функции SWITCH в формуле поля

10. Формула суммы с учетом скидки - задается в параметре "Значение по умолчанию" в свойствах поля. Использование функции IIF в формуле хранимого числового поля

11. Запрос, чтение XML и импорт - демонстрация получения информации о курсах валют через форму и через команду GoToUrlAndImportXml из кода кнопки

12. Облачные диски - демонстрация синхронизации копий баз данных (реплик) через облачные диски

13. Сумма прописью - демонстрация использования функции SpellMoney с разными валютами

14. Экспорт в XML и импорт - демонстрация использования экспорта в XML и последующего импорта
