Страница программы "Учет клиентов"

Конфигурация "Рыболовная ферма" - это универсальная система автоматизации учета для рыболовных хозяйств. Она способствует ведению учёту по наличию рыбы на различных водоёмах.

Программа предусматривает бронирование и аренду домов на определённую дату; учёт путёвок для клиентов и расчёт их стоимости с учётом системы скидок по клубным картам.

При помощи конфигурации "Рыболовная ферма" возможно ведение базы данных клиентов, отслеживание платежей по аренде.

Программа позволяет формировать отчёты по предоставленным услугам и по состоянию водоёмов, а также хранить справочники и создавать новые.

Конфигурация "Рыболовная ферма" даёт возможность хранить информацию о сотрудниках и позволяет настраивать для каждого из них персональные права доступа к данным.



Основные функции:

Учет клиентов
Ведение базы данных клиентов. К каждому клиенту можно
привязать несколько транспортных средств, для учета въезда
на территорию рыболовной фермы.
Учет путевок
Регистрация и учет путевок для клиентов.
Расчет стоимости путевки до рыбалки, и улова после рыбалки.
Возможность расчета с учетом системы скидок по клубным картам.
Аренда домов
Ведение учета аренды мест размещения для клиентов.
Отслеживание платежей по аренде.
Бронирование
Регистрация и контроль забронированных домов.
Складской учет
Ведение учета по приходу и расходу рыбы на различных водоемах.
Учет сотрудников
Хранение информации о сотрудниках, настройка
персональных прав доступа.
Формирование отчетов
Формирование отчетов по предоставляемым услугам
и бронированию домов на дату, а так же по состоянию водоемов.
Возможность создавать новые отчеты.
Хранение справочников
Возможность хранения и создания новых справочников.
